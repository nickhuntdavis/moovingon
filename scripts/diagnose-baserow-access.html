üìã How to Fix "ERROR_USER_NOT_IN_GROUP"

Option 1: Add Token to Workspace (Recommended)

  1. Go to your Baserow workspace
  2. Click on the workspace settings (gear icon)
  3. Go to "API tokens" or "Members" section
  4. Add your token to the workspace, OR
  5. Create a new token that has access to this workspace

Option 2: Create a New Token with Workspace Access

  1. In Baserow, go to your user settings
  2. Navigate to "API tokens"
  3. Create a new token
  4. Make sure it has access to the workspace containing table "MoovingOn 345196"
  5. Update the token in services/baserow.ts

Option 3: Verify Table ID

  1. The table ID might be incorrect
  2. Use "Find Tables" above to see actual table IDs
  3. Update TABLE_ID = '345196' in services/baserow.ts if needed
